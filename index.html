CityGis

CityGis is verantwoordelijk voor de software van de meldkamer in Nederland. Om ervoor te zorgen dat ze de meest geschikte wagen naar de plek van een noodgeval kunnen sturen heeft elke dienstwagen een systeem die vanalles bijhoud over de wagen.

Wij kregen een week aan data van een paar wagens en we moesten met deze ruwe data bruikbare data maken voor bijvoorbeeld de wagenparkbeheerder. We hebben dit project afgesloten met een 9,1.

  • Opdrachtgever: Project van school in samenwerking met CityGis.
  • Project duur: 18 weken
  • Github Link: Project op github

Euromast

Wij moesten voor de euromast in Rotterdam een portal maken om reacties van mensen via social media zichtbaar te maken. Wij moesten bijvoorbeeld twitter en facebook doorzoeken om te kijken of er berichten over de euromast waren. Hierbij moesten wij ook kijken of dit een positief of negatief bericht was.

Dit is het derde project waar ik aan heb gewerkt voor school. We hebben dit project afgesloten met een 8,2.

  • Opdrachtgever: Project van school in samenwerking met de euromast.
  • Project duur: 9 weken
